# Super Tic-Tac-Toe Game Engine

Coding a Super Tic-Tac-Toe game engine to best my students every. single. time.

## Progress

- **Language:** C++17
- **Build System:** CMake
- **AI Algorithm:** Minimax with alpha-beta pruning
